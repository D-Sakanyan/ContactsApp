# Contacts App

A simple CRUD web application for managing contacts built with Spring Boot, JDBC, PostgreSQL, and Thymeleaf.

---

## Tech Stack

* Java 17+
* Spring Boot (MVC)
* JdbcTemplate
* PostgreSQL
* Thymeleaf
* Docker / Docker Compose

---

## Features

* Create, update, delete contacts
* View contact list
* Server-side validation (email, phone)
* Single form for create/update

---

## Run with Docker

```bash id="d1"
docker-compose up --build
```

Open:

```text id="d2"
http://localhost:8080/contacts
```

---

## ⚙️ Run Locally

### 1. Start PostgreSQL

Create database:

```sql id="d3"
CREATE DATABASE contacts_db;
```

Create table:

```sql id="d4"
CREATE TABLE contacts (
    id SERIAL PRIMARY KEY,
    firstName TEXT,
    lastName TEXT,
    email TEXT,
    phone TEXT
);
```

---

### 2. Configure application

```properties id="d5"
spring.datasource.url=jdbc:postgresql://localhost:5432/contacts_db
spring.datasource.username=YOUR_USERNAME
spring.datasource.password=YOUR_PASSWORD
```

---

### 3. Run application

```bash id="d6"
./gradlew bootRun
```

## Endpoints

| Method | URL                   | Description    |
| ------ | --------------------- | -------------- |
| GET    | /contacts             | List contacts  |
| GET    | /contacts/create      | Create form    |
| POST   | /contacts/create      | Save contact   |
| GET    | /contacts/edit/{id}   | Edit form      |
| POST   | /contacts/edit        | Update contact |
| POST   | /contacts/delete/{id} | Delete contact |

---

## Architecture

```
Controller → Service → Repository → Database
```

---

## Notes

* ID is auto-generated by DB
* Validation handled on server side
* Uses JdbcTemplate (no JPA/Hibernate)
